# GitLab Stand - Шаблонный проект

Этот проект служит шаблоном для создания модулей в тестовом стенде.

## Структура

- `pyproject.toml` - конфигурация Python пакета 
- `.gitlab-ci.yml` - пайплайн для сборки и публикации в Nexus
- `src/` - исходный код модулей 

## CI/CD Pipeline

При создании тега (например, `v1.0.0`) автоматически:
1. Собирается wheel пакет (`uv build --wheel`)
2. Публикуется в Nexus (`uv publish`)

Переменные окружения, используемые в пайплайне:
- `NEXUS_URL` - URL Nexus репозитория
- `NEXUS_USER` - имя пользователя для публикации
- `NEXUS_PASSWORD` - пароль для публикации

## Использование

Этот проект автоматически импортируется в GitLab при запуске `setup_dev_env.py`.
